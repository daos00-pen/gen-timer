        <h1>
         <span>
          系統性的安全警示
         </span>
        </h1>
        <div>
         <!--Begin Content-->
         <!-- Start Content -->
         <table>
          <tbody>
           <tr>
            <td>
             <div>
              <img alt="Labour Department Logo"/>
             </div>
            </td>

             <strong>
              職業安全及健康部
              <br/>
              Occupational Safety and Health Branch
              <br/>

              <br/>
              Labour Department
             </strong>

         <p style="text-align: center;">
          <strong>
           <span style="text-decoration: underline;">
            <span style="font-size: 1.125em;">
             系統性的安全警示
            </span>
           </span>
          </strong>
         </p>
         <p>
          承建商/僱主有責任確保其僱用的工人/僱員在工作期間的職業安全及健康，註冊安全主任及註冊安全審核員則須根據《工廠及工業經營(安全主任及安全督導員)規例》及《工廠及工業經營(安全管理)規例》所闡述的法定職責，為僱主/委託人提供意見及所需協助，以便履行其安全責任。你在執行註冊安全主任及/或註冊安全審核員的法定職責時，應充分顧及以下
          <span style="font-weight: bold;">
           六項基本安全管理原則：
          </span>
         </p>
         <ol style="margin-left: 1em;" type="i">
          <li>
           在進行風險評估及制定施工方案、工作系統和相關的危害控制措施時，應充分考慮不同工作的性質、地點、施工方法和工作時使用的機械/設備；
          </li>
          <li>
           應優先在根源消除危害(例如禁止使用有害物質)或採取工程控制措施(例如採用較安全的工作/生產程序)，避免把工人置於潛在危害狀況，其次是考慮可緩減危害的措施;個人防護裝備則屬最後的防護措施；
          </li>
          <li>
           應制定和實施健全的審查制度，確保「合資格人士」和僱主指定的其他人士履行其職責，以確保使用高危機械和進行高危工作時的安全；
          </li>
          <li>
           應制定和實施有效的視察計劃，主動查察違反安全規定的情況，及早向僱主報告並從速改善；
          </li>
          <li>
           應為總承建商、分判商、各級地盤管理/督導人員和工人制定和維持有效的協調/溝通制度，以確保他們清晰理解工作的潛在危害、內部安全規則、危害控制計劃和界定安全責任；以及
          </li>
          <li>
           應充分顧及有關工作的性質、地點、施工方法和工作時使用的機械/設備，及有關工人(例如少數族裔人士)使用的語言，為工人和督導人員提供適切的安全訓練和資料。
          </li>
         </ol>
